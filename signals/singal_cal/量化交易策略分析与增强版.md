## 2. 决策理论基础

### 2.1 价格行为与量价关系（核心理论）

**威科夫（Wyckoff）量价分析**是系统所有策略的底层逻辑：

| 阶段 | 特征 | 对应策略信号 |
|------|------|------------|
| 积累期 | 低量缩量整理、无大阴线 | B1地量条件、SCB地量判断 |
| 启动期 | 放量大阳、突破 | 暴力K、关键K |
| 上涨期 | 量价齐升 | B2放量新高 |
| 派发期 | 高位大量、出现长上影 | S1次高点、S1条件1 |

**理论依据**：
- 量是价的先行指标（Volume precedes Price）
- 地量缩量是主力完成吸筹的标志
- 放量突破是主力开始拉升的信号

### 2.2 KDJ 随机指标（Stochastic Oscillator）

**数学原理**（Lane, 1950s）：

```
RSV(n) = (Close - LLV(Low, n)) / (HHV(High, n) - LLV(Low, n)) × 100
K = EMA(RSV, 3)    # 平滑随机值
D = EMA(K, 3)      # K的移动平均
J = 3K - 2D        # 超买超卖放大器
```

**为何有效**：J 值是对市场超买超卖的线性放大，当 J < 10~20 时，意味着价格处于近期波动区间的极低位置，均值回归概率高。

**系统中的应用**：
- B1：J < 13 为触发前置条件（极度超卖）
- B2：前一日 J ≤ 21 为触发前置条件
- DZ30：短期KD（3日） ≤ 30，长期KD（21日） ≥ 80

### 2.3 RSI（相对强弱指数）

**Wilder（1978）公式**：

```
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - 100/(1+RS)
```

**系统中的多周期RSI**：
- RSI1（14日）：短期超卖信号（< 20~23）
- RSI2（14日前段）、RSI3、RSI4：多周期多头排列判断（RSI2 > RSI3 > RSI4）

**理论依据**：多周期RSI多头排列说明各时间维度的买入力量均强于卖出力量，趋势延续性强。

### 2.4 MACD 与 DIF

**Appel（1979）公式**：

```
EMA12 = 12日指数移动平均
EMA26 = 26日指数移动平均
DIF = EMA12 - EMA26    # 快慢线差值
DEA = EMA(DIF, 9)      # 信号线
MACD = 2 × (DIF - DEA) # 柱状图
```

**系统应用**：DIF ≥ 0 表示短期均线在长期均线上方，多头趋势。DIF 背离（价格创新高但DIF未创新高）是见顶信号。

### 2.5 BBI（多空均衡指标）

```
BBI = (MA3 + MA6 + MA12 + MA24) / 4
```

**理论**：多周期均线等权平均，减少单一周期噪声。BBI > 20日前BBI 说明趋势向上。

### 2.6 知行多空线（原创复合趋势线）

```
知行多空线 = (MA14 + MA28 + MA57 + MA114) / 4
```

**特点**：使用斐波那契比例相关的周期（14:28:57:114 ≈ 1:2:4:8），覆盖短中长期趋势，权重均等。

**知行短期趋势线** = EMA10 的 EMA10（双重平滑指数均线），减少价格噪声，更好追踪短期趋势。

### 2.7 ATR（真实波幅）动态止损

```
TR = max(High-Low, |High-PrevClose|, |Low-PrevClose|)
ATR(n) = SMA(TR, n)
止损价 = 买入价 - k × ATR
```

**Wilder（1978）**：ATR是市场波动性的客观度量，用ATR设定止损能自适应当前市场状态，避免被噪声震出。

## 3. 现有策略深度解析

### 3.1 B1 策略（低量回调买入）

**本质**：在上升趋势中寻找量能萎缩的回调低点。

**触发前置三条件（必须全满足）**：
1. DIF ≥ 0（MACD多头区域）
2. 知行短期趋势线 > 知行多空线（趋势多头）
3. J < 13（KDJ极度超卖）
4. B1总分 ≥ 8

**39个子条件逻辑分层**：

| 层次 | 条件编号 | 核心逻辑 |
|------|---------|---------|
| 趋势确认 | 1,2,7 | MACD多头、价格在均线上方、趋势线关系 |
| 量能分析 | 8-13, 23, 27 | 地量识别、倍量特征、量的趋势 |
| 价格形态 | 3-5, 18-21 | 涨幅限制、振幅、近趋势线 |
| RSI超卖 | 6, 35 | 超卖区域、多头排列 |
| 风险扣分 | 24-25, 28, 31, 34, 36-39 | 振幅过大、跳空、十字星、大长阴 |
| 特殊形态 | 30 | J值高位滞留后小幅调整 |

**胜率提升关键**：条件30（J值高位整理后回调）是高质量信号，得分+2.8分。

### 3.2 B2 策略（超卖反弹买入）

**本质**：在超卖后的第一根强势大阳线买入（动量反转）。

**触发前置（更严格的量价条件）**：
```
前日J ≤ 21 OR 前日RSI1 ≤ 21
AND 今日成交量 > 昨日成交量
AND 今日涨幅 > 3.95%
AND 成交量 > 60日均量
AND 收盘 > 开盘（阳线）
```

**B2的优势**：相比B1，B2的信号更少但更精准，是典型的"质量优先"策略。

### 3.3 SCB 沙尘暴策略

**本质**：地量缩量后的爆量暴力K，是B1的增强确认版。

**逻辑链**：
```
过去1~5天出现地量基础条件（≥30天地量）
AND 今日出现暴力K（涨幅>4%, 放量1.8倍, 上影线短）
AND 知行多空线 > 60日前多空线（长期趋势向上）
```

**理论**：地量是主力完成吸筹的结束信号，暴力K是主力开始拉升的启动信号，二者结合是最高质量的底部反转信号。

### 3.4 DZ30 单针30策略

**本质**：短期超卖极值 + 长期价格高位。

```
长期KD（21日）≥ 80（长期强势）
AND 短期KD（3日）≤ 30（短期超卖）
AND 价格在趋势线上方
AND 近20日有倍量柱（主力活跃）
```

**理论**：这是"强势调整买点"——整体趋势强，短期过度调整，均值回归概率高。

### 3.5 S1 卖出策略

**两大卖出形态**：

1. **条件1（顶部阴线）**：今日阴线 + 近期最高价 + 10日或50日大幅涨幅
2. **条件2（次高点阴线）**：近4日含最高点但今日非最高 + 阴线 + 放量

**多空线跌破**：收盘跌破知行多空线第二日确认卖出（防止当日假破）。

## 5. 增强版指标模块

以下为 `basic_module_v2.py` 核心改进（完整代码见同名文件）：

### 5.1 向量化KDJ计算（性能提升50x）

```python
def calculate_kdj_vectorized(close_arr, high_arr, low_arr, n=9, m=3):
    """
    标准KDJ向量化计算，使用pandas rolling窗口
    相比原始循环版，速度提升约50倍
    """
    import pandas as pd
    s = pd.Series(close_arr)
    h = pd.Series(high_arr)
    l = pd.Series(low_arr)
    
    low_n = l.rolling(n, min_periods=1).min()
    high_n = h.rolling(n, min_periods=1).max()
    
    rsv = (s - low_n) / (high_n - low_n).replace(0, 1) * 100
    rsv = rsv.fillna(50)
    
    k = rsv.ewm(alpha=1/m, adjust=False).mean()
    d = k.ewm(alpha=1/m, adjust=False).mean()
    j = 3 * k - 2 * d
    
    return k.values, d.values, j.values
```

### 5.2 O(n) 知行短期趋势线

```python
def calculate_知行短期趋势线_fast(close_arr):
    """
    O(n)递推EMA计算，替代原O(n³)版本
    alpha = 2/(10+1) = 2/11
    """
    alpha = 2 / 11
    n = len(close_arr)
    
    # 第一步：计算EMA10序列
    ema10 = np.zeros(n)
    ema10[0] = close_arr[0]
    for i in range(1, n):
        ema10[i] = alpha * close_arr[i] + (1 - alpha) * ema10[i-1]
    
    # 第二步：对EMA10再做EMA10
    ema10_2 = np.zeros(n)
    ema10_2[0] = ema10[0]
    for i in range(1, n):
        ema10_2[i] = alpha * ema10[i] + (1 - alpha) * ema10_2[i-1]
    
    return ema10_2
```

### 5.3 新增 ATR 计算

```python
def calculate_atr(high_arr, low_arr, close_arr, period=14):
    """
    真实波幅ATR计算
    用于动态止损和仓位计算
    """
    n = len(close_arr)
    tr = np.zeros(n)
    tr[0] = high_arr[0] - low_arr[0]
    
    for i in range(1, n):
        hl = high_arr[i] - low_arr[i]
        hc = abs(high_arr[i] - close_arr[i-1])
        lc = abs(low_arr[i] - close_arr[i-1])
        tr[i] = max(hl, hc, lc)
    
    # Wilder平滑（等同于EMA with alpha=1/period）
    atr = np.zeros(n)
    atr[period-1] = np.mean(tr[:period])
    for i in range(period, n):
        atr[i] = (atr[i-1] * (period-1) + tr[i]) / period
    
    return atr
```

## 8. 风控模块

### 8.1 ATR动态止损

```python
# risk_module.py
def calculate_dynamic_stoploss(indicators, buy_price, atr_multiplier=2.0):
    """
    基于ATR的动态止损
    
    理论依据（Wilder 1978）：
    ATR代表市场的平均波动幅度
    止损 = 买入价 - k × ATR
    k通常取2~3，平衡被震出和止损过宽的矛盾
    
    优于固定百分比止损的原因：
    - 高波动股票（振幅大）使用更宽止损，避免被噪声震出
    - 低波动股票（振幅小）使用更紧止损，控制亏损
    """
    atr = calculate_atr(
        indicators['high_arr'], 
        indicators['low_arr'], 
        indicators['close_arr']
    )[-1]
    
    stoploss = buy_price - atr_multiplier * atr
    stoploss_pct = (buy_price - stoploss) / buy_price * 100
    
    # 止损不超过8%（硬性上限），不低于1.5%（防止被极低波动率误导）
    stoploss_pct = max(1.5, min(8.0, stoploss_pct))
    stoploss = buy_price * (1 - stoploss_pct / 100)
    
    return stoploss, stoploss_pct, atr
```

### 8.2 仓位管理（Kelly准则简化版）

```python
def calculate_position_size(win_rate, avg_win, avg_loss, total_capital, max_position_pct=0.3):
    """
    Kelly公式仓位计算
    
    理论（Kelly 1956）：
    最优仓位比例 = (胜率 × 赔率 - 败率) / 赔率
    其中赔率 = avg_win / avg_loss
    
    实践中通常用半Kelly（×0.5）降低波动
    """
    if avg_loss == 0:
        return 0
    
    b = avg_win / avg_loss  # 赔率
    q = 1 - win_rate        # 败率
    kelly_f = (win_rate * b - q) / b
    
    # 半Kelly降低风险
    half_kelly = kelly_f * 0.5
    
    # 限制单仓上限
    position_pct = min(half_kelly, max_position_pct)
    position_pct = max(0, position_pct)
    
    return position_pct * total_capital
```

## 9. 市场状态过滤器

### 9.1 大盘环境判断

```python
# market_filter.py
def check_market_condition(index_close_arr, index_volume_arr):
    """
    大盘状态过滤器
    
    理论依据：
    - 个股与大盘高度相关（A股β系数通常0.6~1.2）
    - 熊市中买入信号的胜率比牛市低30~50%
    - 需要根据大盘状态动态调整策略阈值
    
    返回：
        'bull'：多头市场（放宽买入条件）
        'bear'：空头市场（提高买入阈值，减少交易）
        'range'：震荡市场（正常阈值）
    """
    if len(index_close_arr) < 60:
        return 'range'
    
    # 20日涨幅
    ma20 = np.mean(index_close_arr[-20:])
    ma60 = np.mean(index_close_arr[-60:])
    current = index_close_arr[-1]
    
    # 量能趋势
    vol_ratio = np.mean(index_volume_arr[-5:]) / np.mean(index_volume_arr[-20:])
    
    if current > ma20 > ma60 and vol_ratio > 1.0:
        return 'bull'
    elif current < ma20 < ma60 and vol_ratio < 0.8:
        return 'bear'
    else:
        return 'range'

def adjust_threshold_by_market(base_threshold, market_state):
    """根据大盘状态调整买入阈值"""
    adjustments = {
        'bull': -1.0,   # 牛市降低阈值，积极买入
        'range': 0.0,   # 震荡市保持不变
        'bear': +2.0,   # 熊市提高阈值，谨慎买入
    }
    return base_threshold + adjustments.get(market_state, 0)
```

## 11. 策略调参建议

### 11.1 B1策略优化方向

1. **减少39个条件中的冗余扣分**：条件24（区间振幅）中，60%/70%/80%/90%/100%五级扣分共累计-3分，但振幅>100%实际上意味着翻倍行情，不应该一刀切扣分，可以加入"是否已充分调整"判断。

2. **条件30加权**：J值高位滞留后回调（+2.8分）是高质量信号，可以单独作为触发前置条件之一，而不是纯加分项。

3. **时间窗口自适应**：对于科创板/创业板（高波动）股票，波动率阈值应适当放宽（×1.3~1.5）。

### 11.2 B2策略优化方向

1. **修复知行线数组bug**（最高优先级）：`得分14`和`得分30`由于`知行多空线_arr`和`知行短期趋势线_arr`未被计算（初始化为空列表），这两个得分永远为0，导致B2系统性少计约1.5分，当前阈值8分实际等效于9.5分，可能错过一些信号。

2. **前一日KDJ计算精度**：当前近似计算前一日J值时，没有完整的历史K、D状态（直接用RSV当K/D），导致前一日J值计算不准确。建议从`basic_module`预计算完整KDJ序列并传入。

### 11.3 整体系统优化

1. **信号共振评分**：增加一个`composite_score`字段 = B1×0.3 + B2×0.4 + SCB×0.2 + DZ30×0.1，便于综合排序。

2. **回测验证**：在调整任何参数前，先用`backtest`模块跑历史回测，以年化收益/最大回撤/胜率三个指标评估改动效果。

3. **特征重要性分析**：可以把`indicators` JSON字段与3~10日后涨跌幅做相关性分析，找出历史上最有预测力的特征，反向验证各条件权重是否合理。
